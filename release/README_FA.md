## ⚡ نصب سریع

### 🌐 سرور آنلاین (به اینترنت وصل است)

یک دستور را کپی کنید، در ترمینال سرور بزنید، تمام:

```bash
cd /tmp && curl -fsSL https://github.com/callmedimi/ElementX/archive/refs/heads/main.tar.gz | tar -xz && cd ElementX-main/release/online && sudo ./install_server.sh
```

> اسکریپت از شما می‌خواهد دامنه یا IP، و نام کاربری ادمین را وارد کنید. بقیه را خودش انجام می‌دهد.

---

### 🔌 سرور آفلاین (بدون اینترنت)

**مرحله ۱ — روی کامپیوتر شخصی (باید اینترنت داشته باشد):**

```bash
# دانلود ریپوزیتوری و آماده‌سازی فایل‌ها
cd release/offline

# ویندوز:
prepare_assets.bat

# لینوکس / مک:
bash prepare_assets.sh
```

**مرحله ۲ — پوشه `offline/` را کامل به سرور منتقل کنید (USB یا شبکه داخلی)**

**مرحله ۳ — روی سرور:**

```bash
cd offline
sudo ./install_server.sh
```

## ⚙️ آدرس‌های مهم بعد از نصب

| آدرس | کاربرد |
|---|---|
| `https://yourdomain.com/register` | صفحه ثبت‌نام کاربران جدید |
| `https://yourdomain.com/admin` | پنل مدیریت (کاربران و اتاق‌ها) |
| `data/homeserver.yaml` | فایل تنظیمات Synapse |

---

## 🛠️ مدیریت سرور با `manage.sh`

داخل پوشه نصب روی سرور اجرا کنید:

```bash
sudo ./manage.sh
```

| گزینه | کار |
|---|---|
| `1` | ساخت کاربر جدید |
| `2` | مشاهده لاگ‌های زنده Synapse |
| `3` | راه‌اندازی مجدد سرویس‌ها |
| `4` | بررسی وضعیت کانتینرها |
| `5` | روشن/خاموش کردن ثبت‌نام عمومی |
| `6` | ⚠️ تعمیر سریع — اصلاح تنظیمات و دسترسی‌ها |

---

## 🆘 مشکلات رایج

**سرور راه‌اندازی نمی‌شود یا خطای ۵۰۲ می‌دهد؟**
```
sudo ./manage.sh  →  گزینه 6 (تعمیر سریع)
```

**صفحه ثبت‌نام حتی بعد از غیرفعال کردن هنوز فرم نشان می‌دهد؟**
```
sudo ./manage.sh  →  گزینه 5 (صفحه به صورت خودکار بروزرسانی می‌شود)
```

**تنظیمات Synapse خراب شد؟**
```
گزینه 6 پشتیبان خودکار می‌گیرد و تنظیمات را بازیابی می‌کند.
```
